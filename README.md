To run project: 

EITHER: \
Run this command in the root folder:
```
mvn javafx:run -f ui/pom.xml
```

OR:

- Change directory to [module-template/ui](/chore-manager/ui/)
- Run javafx run command
```
cd chore-manager/ui/
mvn javafx:run
```
